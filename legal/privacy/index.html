隐私 ​

以下内容适用于本站和 vanadiry.com 下的所有子域，下文简称“本站及其子域”。

本页面的其他语言版本均由机器翻译生成，仅供参考。如有歧义，应以中文版本为准。

上次修改：2025年3月13日

访问统计 ​

本站及其子域，均使用 Microsoft Clarity 收集你访问本站的统计数据，例如你访问的页面和在每个页面上的停留时间。

收集的数据仅用于改进用户体验，所有信息均经过匿名处理，并会在 3 个月后删除。
Vanadiry 承诺不会将数据出售或分享给任何第三方。

Microsoft 是统计数据的提供商。

评论系统 ​

magic.vanadiry.com 使用的讨论系统将数据储存在专门的仓库，发表评论之前，你需要使用 GitHub 登录，你发表的评论将会公开可见。

GitHub 是评论数据储存的提供商。

第三方服务商 ​

本站使用的服务商也可能会收集你的信息，请参考服务商的隐私政策：

  • Cloudflare
  • Vercel
  • Microsoft
  • GitHub
  • Google

联系方式 ​

如果你对本隐私政策或个人信息保护相关事宜有任何疑问、建议的，可以通过邮件联系我：v@vanadiry.com。
